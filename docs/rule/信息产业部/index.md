## 简介

中华人民共和国信息产业部成立于1998年3月，2008年3月15日 并入中华人民共和国工业和信息化部。 

+   先前机关
    +   中华人民共和国电子工业部
    +   中华人民共和国邮电部
    +   中华人民共和国广播电影电视部（部分）
+   后继机关
    +   并入中华人民共和国工业和信息化部
