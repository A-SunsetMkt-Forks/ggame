## 后果

+ 2021年9月初，百度贴吧取消了「傻逼」一词的封锁
+ 哔哩哔哩开展了针对光荣职业污名化的集中治理，将处理「伞兵」「nmsl」「孝子」「孤儿」等词语[^781544]
+ 网易游戏将伞兵添加进了用户名敏感词系统，这将无法使用含有伞兵的字符作为用户名[^ne_bw]
