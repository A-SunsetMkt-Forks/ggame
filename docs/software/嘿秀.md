## 被处罚

> 近日(2016年7月12日 左右)，直播平台“嘿秀”因传播淫秽、色情直播内容被北京市文化市场行政执法总队（以下简称市文化执法总队）及北京市公安局网安总队责令停业整顿。该直播平台因未取得《网络文化经营许可证》，被市文化执法总队给予行政罚款。[^n459]
