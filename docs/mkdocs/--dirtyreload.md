## 加速生成

使用 `--dirtyreload` 这个参数可以加速生成，因为现在我这里已经有超过 500 个 markdown 页面，所以生成时间达到了 20 秒，调试起来根本就不是实时的，所以需要使用这个参数。

```shell
mkdocs serve --dirtyreload
```
