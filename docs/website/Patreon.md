## DNS 污染

2022年6月2日，多个 Patreon 用户表示无法连接网站，[^268067324][^1494875]随后在 6月4日 有人测试了 DNS 解析，发现部分地区将其解析到了 Facebook 的 IP 地址。[^857266]
