# 皇后大道东

## 简介

《皇后大道东》是罗大佑创作的，带有政治隐喻的歌曲。

## 被下架

2019年6月19日 左右，QQ音乐、[网易云音乐][]、酷狗音乐和虾米音乐都无法找到《皇后大道东》这首歌了，罗大佑的另一首歌《东方之珠》也被限制了评论。[^2024914]
