2022年7月8日，<ruby>梁靜茹<rt>梁静茹</rt></ruby>的《可惜不是你》遭到了审查，一开始被下架了一段时间，[^1545]

[^1657]: 《[梁靜茹歌曲《可惜不是你》突遭內地平台下架](https://web.archive.org/web/20220709105738/https://ol.mingpao.com/ldy/showbiz/latest/20220709/1657347488964/梁靜茹歌曲《可惜不是你》突遭內地平台下架)》, 明報OL網, 2022-07-09. (参照 2022-07-09).

[^1545]: 推特小红旗, 《[7月8日，日本前首相安倍晋三遇刺，由歌手梁静茹演唱的《可惜不是你》被紧急下架，原因不明。](https://web.archive.org/web/20220708085530/https://twitter.com/Xhnsoc__Redflag/status/1545287201352830978)》, Twitter, 2022-07-08. (参照 2022-07-09).

### 叮咚鸡

「叮咚鸡」是深圳社区工作人员在回答隔离相关问题时，用方言快速回答 「等通知」所产生的空耳。[^41721]
这其间还出现了「大狗叫」「戴口罩」、「兴奋劲」「身份证」、「带鼠鸡」「带手机」这样的空耳。

[^41721]: 今日热点网, 《[叮咚鸡是什么梗？叮咚鸡是什么意思？](https://web.archive.org/web/20250701142156/http://www.zhoukan.cc/keji/2025/0508/341721.html)》, 环球周刊网, 2025-05-08. (参照 2025-07-01).

2022年8月26日，似乎是为了宣传即将在 10月1日 发布的动画《大白日记》，所以哔哩哔哩 UP 主「大白小天使」发布了歌曲《叮咚鸡》，
原标题为《动漫版海南魔性“叮咚鸡”来啦！》。

《大白日记》似乎是对抗《方方日记》的作品，简介为「以疫情期间，发生在那些防疫志愿者身上的某些暖心瞬间为创作背景，
通过动画形式展现在大众眼前。」

然而《大白日记》及其《叮咚鸡》在解封后的未知时间，被限制。前者的哔哩哔哩动画页面 URL 无法访问，[^39707]
后者无法通过搜索找到，只能通过 URL [BV1Ee4y1Z7NZ](https://www.bilibili.com/video/BV1Ee4y1Z7NZ) 或 UP 主的主页进入。

[^39707]: 游洋, 《[大白日记](https://web.archive.org/web/20230315034036/https://www.bilibili.com/bangumi/media/md28339707)》, 哔哩哔哩弹幕视频网, 2022-10-01. (参照 2025-07-01).
