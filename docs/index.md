## 搜索功能

MkDocs 的搜索功能暂时不太好用，所以 /punish 和 /news_archive 板块的绝大部分内容都被排除了。因为搜索功能的本质是一个包含所有内容的 json 文件，大小为 2 MB 左右，而这些部分的内容只是资料库，并且量大。

更好的中文搜索预计 Material for MkDocs 更新到 10.0.0 版本时，就能够使用了。
