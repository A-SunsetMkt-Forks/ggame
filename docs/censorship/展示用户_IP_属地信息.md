## 简介

2021年10月底，中国网信网发布了《[互联网用户账号名称信息管理规定（征求意见稿）](/rule/国家互联网信息办公室/互联网用户账号名称信息管理规定（征求意见稿）.md)》。

里面的第十二条提到了「互联网用户账号服务平台应当以显著方式，在互联网用户账号信息页面展示账号IP地址属地信息。境内互联网用户账号IP地址属地信息需标注到省（区、市），境外账号IP地址属地信息需标注到国家（地区）。」

+   相关讨论
    +   《[如何解读网信办意见稿第十二条：应以显著方式在用户账号信息页面展示用户IP属地信息？](https://web.archive.org/web/20211228115535/https://www.zhihu.com/question/494688998)》, 知乎, 2021-10-26. (参照 2022-04-18).

不过直到 2022年4月18日，此规定依然没有发布正式文件。

## 执行

2022年3月开始，部分社交软件开始执行这项未发布正式文件的规定了。

???+ quote "当前俄乌局势严峻，站方在持续深入对相关不良信息进行排查处置的同时……"

    当前俄乌局势严峻，站方在持续深入对相关不良信息进行排查处置的同时，发现存在极个别冒充在乌人员发布不实言论的行为。为了在重大事件中维护良好传播秩序，站方拟于近期上线针对特定用户展示发博、发评论真实地理位置功能， 具体说明如下：

    1. 此功能目前仅作为俄乌冲突时期特定内容（例如：博文中带有“俄罗斯”、“乌克兰”、“基辅”等关键词）的地域展示，以便更透明地向广大用户呈现真实情况。
    2. 站方将权衡用户体验，仅在有必要时，对部分用户展示微博/评论地理位置信息（国内显示到省份/地区，国外显示到国家），该功能暂时无法由用户主动开启或关闭。
    3. 站方将根据内部测试情况，在未来几天逐步上线相关功能。

    2. 评论展示真实地理位置信息
