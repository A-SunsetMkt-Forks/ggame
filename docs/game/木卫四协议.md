《木卫四协议》是一款剧情推动型第三人称生存恐怖游戏，故事时间背景设定为300年后。 玩家将扮演雅各布·李，他受命运的捉弄被关进了黑铁监狱，这是一座位于木星卫星木卫四中安全等级最高的监狱。 在同狱犯人变成极为可怕的怪异生物那一刻起，整个监狱便陷入了混乱。
