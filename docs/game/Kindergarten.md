## 封禁视频

2019年上半年的某个时候，哔哩哔哩封禁了《Kindergarten》这款游戏，下架了几乎全部的游戏视频。之后腾讯视频等平台也对该游戏进行了封禁，包括续作《Kindergarten2》。
