## 皇家版在美国被审查

2020年2月18日，由于涉及丑化、刻板印象的使用了两个具有刻板印象的女性动作与部分装束的常见男性体格的路人角色，并且剧情中作为反面人物，所以美国的本地化团队打算修改这个剧情。
