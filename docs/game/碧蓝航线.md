《碧蓝航线》是一款由伤海蛮啾网络科技与厦门勇仕网络技术制作，由 bilibili 游戏发行的萌拟人化即时海战类的 iOS 及 Android 手机游戏，于 2017年5月25日 开始营运。日语版由上海悠星（Yostar）代理发行，2017年9月14日 开始营运。港澳台正体中文版由品玩邦艺术代理，于 2019年10月4日 起正式营运。2021年3月2日，品玩邦宣布台港澳的代理权于同年 4月1日 转移给NIJIGEN，并表示玩家的权益不会受到任何影响。[^wiki]

## 官方反和谐

曾经在《碧蓝航线》哔哩哔哩服务器的世界聊天频道里输入：「AF可能缺少淡水」，就可以实现 [官方反和谐](/theme/游戏官方纵容甚至支持的反和谐.md) 效果，让重樱阵营船只名称变为真实舰船名称，但此方法已在 2019 年之前的某个时间失效了。[^af260]

[^af260]: 《[碧蓝航线AF可能缺少淡水](https://web.archive.org/web/20230324142151/https://wiki.biligame.com/blhx/index.php?title=碧蓝航线AF可能缺少淡水&oldid=260)》, 碧蓝航线WIKI, 2019-10-07. (参照 2023-03-24).

另一个方法是修改游戏文件夹里的配置文件，路径和原始内容如下：

```file_path
Android/data/com.bilibili.azurlane/files/localization.txt
```

```ini
Localization = false
Localization_skin = false
```

可以看出这个文件明明像 .ini 文件的格式，却用了 .txt 的扩展名，以及内容简单易懂，可以编辑，
所以这是官方留下的反和谐功能，只要修改 false 值为 true，就能开启，但效果有限。最早应该能还原重樱阵营船的名称，
后来只能实现部分立绘还原了。[^tv5go][^lsalu]

[^tv5go]: -\_-无双-\_-, 《[碧蓝航线反和谐](https://archive.is/tv5go "http://www.bilibili.com/read/cv12321341")》, 哔哩哔哩, 2021-07-26. (参照 2023-03-24).

[^lsalu]: LmeSzinc, 《[碧蓝航线反和谐 AzurLaneUncensored](https://github.com/LmeSzinc/AzurLaneUncensored)》,. 2023-03-24. 参照: 2023-03-24. [Online].

## 斯肯索普问题

2018年5月，日本运营的版本出现了奇怪的问题，将「<ruby>エセックス<rp>(</rp><rt>埃塞克斯</rt><rp>)</rp></ruby>」与「<ruby>サセックス<rp>(</rp><rt>苏塞克斯</rt><rp>)</rp></ruby>」重命名之后，无法再回到原始的名称，此时游戏会提示「エラー2013：NGワードが含まれています」，原因应该是触发了「<ruby>セックス<rp>(</rp><rt>Sex</rt><rp>)</rp></ruby>」这个敏感词。[^gw] 因为这属于敏感词过滤系统过滤超出了本意，所以属于 [斯肯索普问题](/theme/斯肯索普问题.md)。

[^gw]: 《[コメント/サセックス](https://web.archive.org/web/20221014122318/https://azurlane.wikiru.jp/index.php?コメント%2Fサセックス)》, アズールレーン(アズレン)攻略 Wiki. (参照 2022-10-14).

## 手势争议

> [!info inline end]+ 专辑封面
>
> ![vol6_专辑封面](https://s3.tebi.io/ggame/ShareX/game_碧蓝航线_Azur_Lane_vol6_专辑封面.jpg)

2019年11月4日，《碧蓝航线》在推特发布了角色歌专辑《TVアニメーション『アズールレーン』キャラクターソングシングル Vol.6 ユニコーン》的预告，[^31808] 但专辑封面出现了相对少见的手势，与「五大诉求，缺一不可」的手势有些相似，于是有人认为这是支持「反对逃犯条例修订草案运动」。[^41052]

[^31808]: TVアニメ『アズールレーン』, 《[キャラソンCDジャケット公開！](https://web.archive.org/web/20211026115411/https://twitter.com/azurlane_anime/status/1191166773283831808)》, Twitter, 2019-11-04. (参照 2023-03-24).

[^41052]: 餅乾, 《[傑出的兩手〈碧藍航線〉角色手勢被解讀成「五大訴求」，官方圖解滅火](https://web.archive.org/web/20220715195729/https://www.4gamers.com.tw/news/detail/41052/azur-lane-hand-gesture)》, 4Gamers, 2019-11-08. (参照 2023-03-24).

随后《碧蓝航线》官方推特发布了解释，这是日本的数字 6 的手势，与任何特定的政治主张无关，并发布了限制规则，
禁止将官方内容与特定政治主张联系起来。[^97184]
