## 修改游戏内容

《生化危机3重制版》在正式发售前，在 2020年3月19日 推出了 DEMO 演示版的游戏，该演示版仅有 5 分钟，在其中有一幅广告海报，这张海报底部的电话号码被红色的笔圈了三个圈，圈中的数字按顺序分别是「9」「1」「8」。随后有部分玩家在 Steam 评论等地方请愿修改这个解谜的部分。

最后卡普空在正式发售的版本中修改了游戏内容，将数字修改为了「9」「3」「7」。
