???+ quote "9月15日 18:00 更新公告"

    亲爱的各位御主：

    由于技术升级需要，我们将于9月15日18:00起暂时关闭官网及外部渠道的客户端下载入口，给各位御主带来的不便，我们深表歉意。下载入口的关闭不会影响已下载的游戏客户端，已经下载游戏客户端的御主可正常体验游戏。待技术升级结束后，届时御主可恢复游戏客户端的下载。

    感谢各位御主一直以来的理解和支持！[^HIdiL]
