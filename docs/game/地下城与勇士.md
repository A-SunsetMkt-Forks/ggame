## 简介

《地下城与勇士》是韩国 Neople 公司的网络游戏，腾讯代理了大陆简中服务器，最早于 2005年8月 在韩国运营。

## Duel 锁区

2022年6月28日，《DNF Duel》登录 Steam，禁止大陆帐户和 IP 购买，但游戏却含有简体中文等四种语言。

## 一字架

2022年10月27日，腾讯运营的《地下城与勇士》先遣服体验服更新了一些内容，修改了大量素材，涉及英文、宗教、尖牙利齿等，部分红色被限制的感觉，但也有本来不是红色的素材，变成了红色，所以没法确定这次的规避审查，是否主要是 [避开红色](/unclear/禁止人外.md)。[^2247655286]

[^2247655286]: 隔壁的小红花, 《[’DNF’这个名字用了10多年，居然要变了！](https://freewechat.com/a/MzIxMzQ3NjEzOQ==/2247655286/1 "https://mp.weixin.qq.com/s/9b-AFQkYfkBa_YeJbk8ZiQ")》, 微信公众号, 2022-10-27. (参照 2022-10-29).

游戏图标「DNF」变成了「地下城与勇士 创新世纪」，而代币、积分单位 P 和 M 变成了「代」和「积」，本来是红色主题的副本界面更换了主题色，这点可能属于避开红色。然后是十字架被换成一字架，各种宗教元素被移除，技能图标中的骷髅也被移除。

备注：《地下城与勇士：创新世纪》是 2014 年送审的名称。[^15770]

[^15770]: 《[进口网络游戏审批信息-2014年进口网络游戏审批信息](https://web.archive.org/web/20221102012529/https://www.nppa.gov.cn/nppa/contents/318/15770.shtml)》, 国家新闻出版署, 2015-01-23. (参照 2022-11-02).

## 大型和谐版本

2023年2月17日，腾讯表示为了落实主体责任，给用户更加积极向上的内容和服务，进行了部分内容调整。将在 2023年2月23日 的版本更新中实装。[^30266] 这意味着之前体验服的内容，会实际装入到正式版里。

[^30266]: 《[关于游戏名称及部分游戏内容调整的说明](https://web.archive.org/web/20230218113718/https://dnf.qq.com/webplat/info/news_version3/119/495/498/m21449/202302/930266.shtml)》, 地下城与勇士 DNF 官方网站, 2023-02-17. (参照 2023-02-20).
