## 修改女性角色表情 BUG

2021年11月中旬，《哈利波特：魔法觉醒》出现了一个「[阿嘿颜](https://zh.wikipedia.org/zh-hans/阿嘿顏)」事件，许多女性角色受到攻击时可能产生类似「阿嘿颜」。随后游戏在 11月15日，通过关闭面部表情系统的方式来修复 BUG。
